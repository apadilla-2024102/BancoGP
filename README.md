# BancoGP

Este proyecto consiste en el diseño y desarrollo de una plataforma bancaria digital orientada a optimizar la gestión de cuentas y transacciones financieras de forma segura y organizada

Porposito del proyecto
La finalidad es permitir que las operaciones bancarias (depósitos, transferencias, retiros) se realicen de manera estructurada, reduciendo errores y garantizando la protección de la información mediante controles de acceso y encriptación de datos.

Tecnologías utilizadas
El sistema está construido utilizando un stack moderno para garantizar escalabilidad y rendimiento:
    -Backend: Node.js y .Net
    -Frontend: React.
    -Base de Datos: MongoDB.
    -Metodología: SCRUM

Microservicios
El sistema se divide en 5 unidades funcionales independientes para mejorar la administración y seguridad:

    -Autenticación y Control de Acceso: Manejo de sesiones y encriptación de contraseñas.
    
    -Gestión de Clientes y Cuentas: CRUD de usuarios y consulta de saldos.

    -Transacciones: Procesamiento de depósitos, retiros y validación de fondos.

    -Productos Financieros y Divisas: Administración de tasas de interés y conversión de moneda.

    -Reportes y Notificaciones: Generación de estados de cuenta y estadísticas.
